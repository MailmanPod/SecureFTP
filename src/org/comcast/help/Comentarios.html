Para el manejo de la planificacion de las tareas se utiliza la libreria Quartz.

El programa utiliza la libreria TIKA para determinar el tipo de archivo MIME.

Sobre la encriptacion, su utiliza la tecnica simetrica (AES) para encriptar todo el contenido del archivo, y luego se utiliza la tecnica asimetrica (RSA 1024) para realizar un Wrapped sobre la llave simetrica (AES).

El manejo de xml, lo realizan los modulos DOM del paquete de JAVA.

Para crear los wizards que se encargan de subir y descargar archivos, se utilizo la libreria wizard desarrollada por JAVA.

Apache nos proveyo de las dependencias commons.net y commons.codec y commons.io, que se utilizaron para el manejo de la conexion hacia el servidor ftp y para realizar formateo de los datos a presentar al usuario.

La dependencia smtp es la encargada de realizar las transacciones hacia el smtp de Gmail.
